# 国际化模块

## 概述

国际化模块为 QwenPaw ChatUI 提供多语言支持，使用 vue-i18n 库实现。

## 职责

1. 管理翻译文件
2. 提供语言切换功能
3. 集成 Nuxt UI 的 locale prop
4. 支持 URL 参数切换语言

## 使用方法

### 在组件中使用翻译

```vue
<script setup lang="ts">
import { useI18n } from 'vue-i18n'

const { t } = useI18n()
</script>

<template>
  <button>{{ t('chat.send') }}</button>
</template>
```

### 切换语言

```typescript
import { useI18n } from '../composables/useI18n'

const { setLocale } = useI18n()
setLocale('en') // 切换到英文
```

### URL 参数切换

```
http://localhost:3000/?lang=en
```

## 翻译文件结构

```
src/locales/
├── zh-CN/
│   ├── common.json      # 通用文本
│   ├── settings.json    # 设置文本
│   ├── chat.json        # 聊天文本
│   └── components.json  # 组件文本
├── en/
│   ├── common.json
│   ├── settings.json
│   ├── chat.json
│   └── components.json
└── index.ts
```

## 添加新语言

1. 在 `src/locales/` 下创建新语言目录（如 `ja/`）
2. 复制现有翻译文件并翻译
3. 在 `src/locales/index.ts` 中导入新语言
4. 在 `src/composables/settings/definitions.ts` 中添加语言选项
