集成方式

1. 设置系统集成

  • 新增appearance.language分组
  • 语言选择下拉框
  • URL参数lang=?支持

2. 数据库存储

  • 设置键: appearance.language.locale
  • 值: 'zh-CN' | 'en'

3. Nuxt UI集成

  • 使用@nuxt/ui/locale
  • 动态切换locale prop
  • 支持50+语言

组件迁移策略

1. 现有组件改造

// 改造前
<button>发送</button>

// 改造后
<button>{{ $t('chat.send') }}</button>

2. 日期本地化

// 使用dayjs处理日期格式
import dayjs from 'dayjs'
import 'dayjs/locale/zh-cn'
import 'dayjs/locale/en'

function formatDate(date, locale) {
  return dayjs(date).locale(locale).format('YYYY-MM-DD')
}
